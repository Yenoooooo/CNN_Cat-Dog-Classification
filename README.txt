基于卷积神经网络的猫狗图像分类（MATLAB实现）

一、项目简介
本项目为《人工智能》课程算法复现配套代码，基于MATLAB深度学习工具箱从零搭建自定义卷积神经网络，实现猫狗图像二分类任务。
项目完整覆盖数据集加载、随机划分、数据增强、网络搭建、模型训练、性能评估、结果可视化与错误案例分析全流程，可作为深度学习图像分类入门的实践参考。

二、环境要求
- 软件版本：MATLAB R2024b 及以上版本（推荐 R2024b）
- 必备工具箱：
  - Deep Learning Toolbox（深度学习工具箱）
  - Image Processing Toolbox（图像处理工具箱）
- 硬件环境：
  - 推荐：支持CUDA的NVIDIA显卡，自动启用GPU加速训练

四、使用说明

1. 数据集说明
采用公开Kaggle Cats and Dogs猫狗图像数据集，本项目使用了其中猫、狗图像各5000张进行模型训练。
1. 准备猫狗分类图像数据集，按类别分别放入 `CatDogs/cat` 和 `CatDogs/dog` 两个子文件夹中，文件夹名称将自动作为样本标签
2. 支持 jpg、png 等常见图像格式，无需预先统一尺寸，代码会自动将图像缩放至网络输入尺寸

2. 运行步骤
1. 打开 MATLAB，将工作目录切换至项目根目录
2. 打开 `cat_dog_classification.m` 脚本文件
3. 直接运行脚本，程序将自动依次执行以下流程：
   - 加载数据集并按 8:2 比例随机划分训练集、测试集
   - 对训练集施加随机水平翻转数据增强
   - 搭建卷积神经网络并启动训练
   - 在测试集上完成性能评估
   - 生成混淆矩阵与错误案例可视化结果

3. 输出说明
程序运行完成后，将输出以下结果：
- 命令行输出：训练集与测试集样本量、训练总耗时、测试集整体准确率、各类别精确率/召回率/F1值、混淆矩阵数值
- 图形窗口1：训练进度图，包含训练准确率曲线与训练损失曲线
- 图形窗口2：测试集混淆矩阵，直观展示分类对错分布
- 图形窗口3：典型错误案例展示，标注真实标签、预测标签与分类置信度

五、网络结构
网络输入为 128×128×3 的RGB彩色图像，整体采用模块化堆叠设计，参考VGG网络的层级构建思路，具体结构如下：

1. 输入层：接收 128×128×3 尺寸的图像输入
2. 卷积模块1：2层3×3卷积（32通道）+ 批归一化 + ReLU激活 + 2×2最大池化（步长2），输出尺寸 64×64×32
3. 卷积模块2：2层3×3卷积（64通道）+ 批归一化 + ReLU激活 + 2×2最大池化（步长2），输出尺寸 32×32×64
4. 卷积模块3：2层3×3卷积（128通道）+ 批归一化 + ReLU激活 + 2×2最大池化（步长2），输出尺寸 16×16×128
5. 分类器：全连接层（256神经元）+ ReLU激活 + Dropout层（失活率0.5）+ 全连接层（2神经元）+ Softmax层 + 分类输出层

网络通过批归一化加速收敛，结合Dropout与L2正则化抑制过拟合，在参数量适中的前提下保证分类性能。

六、训练参数设置

| 参数项             | 参数值           | 说明                                             |
|---------------|---------------|---------------------------------|
| 优化器             | Adam            | 自适应学习率优化算法               |
| 初始学习率      | 0.0003           | 训练初始阶段的学习率                |
| 学习率策略      | 分段衰减        | 每8轮学习率衰减为原来的0.5倍 |
| 最大训练轮次  | 30轮               | 完整训练的总epoch数               |
| 批次大小         | 128                 | 单次迭代的样本数量                  |
| L2正则化系数 | 0.0001             | 权重衰减约束，抑制过拟合       |
| 梯度阈值         | 1                     | 梯度裁剪阈值，防止梯度爆炸    |
| 数据增强         | 随机水平翻转 | 仅作用于训练集，提升泛化能力 |

七、实验结果
在标准猫狗分类数据集上的测试结果如下：
- 测试集整体分类准确率：87.5%
- 猫类：精确率 88.9%，召回率 86.6%，F1值 87.7%
- 狗类：精确率 86.2%，召回率 88.6%，F1值 87.4%
- 单GPU（RTX4050）环境训练总耗时：约 8分40秒

2. 请确保数据集中无损坏、无法读取的图像文件，避免程序运行中断
3. 本项目为课程学习复现用途，数据集请使用公开合规的公开数据集
